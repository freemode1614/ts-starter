# ts-starter

The project provides a set of tools to simplify your TypeScript development workflow, including dependency management, code formatting, testing, and publishing.

## Usage

1. Install Dependencies:

- Clone this repository.

```bash
pnpm install
```

2. Run Development:

```bash
pnpm dev
```

3. Run bundle:

```bash
pnpm build
```

4. Run test cases:

```bash
pnpm test
```
